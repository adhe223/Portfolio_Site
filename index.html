The Fantasy HOF

This is a purely client side web app that will build a statistical record book for your fantasy football league. The page is all javascript and uses AJAX requests in conjunction with YQL (Yahoo Query Language) to scrape the ESPN fantasy league's page. It displays graphs of the data using the CanvasJS library to provide an easy to understand view of your league's history. Has over 2000 unique visitors and 3000 visits.

NBA Stream Linker

This was my first foray into Node.js + Express and EJS was used for HTML templating. This web application scrapes NBA.com to get the games being played that day. The user provides a list of stream links for each particular team in a text file. When the user clicks on one of the games it will redirect them to the link they provided. BusAlert

Bus Alert is a mobile app designed for the bus riders of Lexington, KY. We wanted to give bus riders with iOS/Android smartphones the power to see the status of a bus stop on the go. We did this by hooking into the GPS data that is present on all Lextran buses in Lexington via the Lextran API. We offer a dynamic map that shows the bus stops on a route and the estimated times of arrival for those stops. On top of that, we have implemented a simple notification system that bus riders can use to alert them with information for a stop at a specified time.

WikiLocale

WikiLocale was my first attempt at mobile development. The goal of the project was to make an app that was useful for me and in the process build valuable, mobile development skills as well as accrue more experience in Java. The app detects when the user is traveling away from home and sends them a notification that links to the Wikipedia page of the city they are currently in. It tests for various situations where it won't fire a notification based on the user's home location (user defineable) or distance traveled since last update. Written in Java using Eclipse and Google's Android SDK.
